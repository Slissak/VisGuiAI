Step-by-step guide generation API with progressive disclosure and adaptive guide generation. This API enables conversion of natural language instructions into structured, actionable guides that reveal steps progressively to avoid overwhelming users.

instruction-guides

API endpoints for instruction-based guide generation and progressive step management

Generate step-by-step guide from natural language instruction

This endpoint uses an LLM to convert natural language instructions into structured, actionable step-by-step guides. The guide uses progressive disclosure - only the first step is returned to avoid overwhelming users.

Authorizations:
BearerAuth
Request Body schema: application/json
required
instruction
required
string

Natural language description of the task

difficulty
string
Default: "beginner"
Enum: "beginner" "intermediate" "advanced"

User's skill level

format_preference
string
Default: "detailed"

Level of detail desired (detailed, concise)

Responses

Local development server

http://localhost:8000/api/v1/instruction-guides/generate
